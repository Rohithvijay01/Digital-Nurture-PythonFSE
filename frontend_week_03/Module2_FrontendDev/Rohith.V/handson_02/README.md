# Responsive Student Portal Layout

## Overview

This HandsOn_02 project extends a Student Portal page with a responsive layout. It provides a portal header, student statistics, and a course catalog designed to adapt to different screen sizes. The work focuses on responsive HTML and CSS implementation.

## Features

- Responsive portal navigation and menu control
- Student statistics for enrolled courses, GPA, and semester
- Five-course catalog with credit information

## Tech Stack

- HTML5
- CSS3

## Project Structure

```text
handson_02/
├── index.html    # Portal content and layout
└── styles.css    # Responsive styles
```

## Installation

```bash
git clone <repository-url>
cd Module2_FrontendDev/Rohith.V/handson_02
```

Open `index.html` in a web browser.

## Usage

View the portal in a browser and resize the viewport to review the responsive layout. The displayed statistics and courses are static content.
