# handson_03

Enhanced Django project with improved REST features.

This folder contains the Week 02 Hands-On 3 Django project. It builds on the previous Django REST API example to add enhancements such as stronger validation, endpoint improvements, and better API responses.

## Contents
- `manage.py` - Django CLI entrypoint
- `db.sqlite3` - SQLite database file
- `coursemanager/` - Django project settings and URL configurations
- `courses/` - enhanced REST API app with models, serializers, and views
- `requirements.txt` - dependencies for Django and DRF
