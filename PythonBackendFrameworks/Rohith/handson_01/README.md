# handson_01

Django course manager application.

This folder contains the Week 02 Hands-On 1 Django project, including a simple course manager built with Django, SQLite, and the `courses` app. It demonstrates Django app structure, models, views, admin configuration, and basic CRUD functionality.

## Contents
- `manage.py` - Django management entrypoint
- `db.sqlite3` - local SQLite database
- `coursemanager/` - Django project settings, URLs, and WSGI/App configuration
- `courses/` - Django app with models, views, and admin setup
- `requirements.txt` - packages required to run this project
