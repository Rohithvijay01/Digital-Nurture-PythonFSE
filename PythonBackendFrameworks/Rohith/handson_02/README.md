# handson_02

Django REST API for course management.

This folder contains the Week 02 Hands-On 2 Django REST Framework project. It extends the course manager into an API-first application with serializers, API views, and REST endpoints for course data.

## Contents
- `manage.py` - Django management command wrapper
- `db.sqlite3` - application database
- `coursemanager/` - project configuration and routing
- `courses/` - app implementation with serializers and REST views
- `requirements.txt` - required Django and DRF packages
