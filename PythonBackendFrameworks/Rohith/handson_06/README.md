# handson_06

FastAPI course API with SQLite.

This folder contains the Week 02 Hands-On 6 FastAPI project. It implements course management APIs using FastAPI, SQLAlchemy, Pydantic schemas, and SQLite.

## Contents
- `main.py` - FastAPI application entrypoint
- `database.py` - database connection setup
- `models.py` - SQLAlchemy models
- `schemas.py` - Pydantic request/response schemas
- `fastapi_courses.db` - local SQLite database file
- `requirements.txt` - FastAPI and SQLAlchemy dependencies
